# geotoolz

> Composable Operator library for remote sensing on top of `georeader.GeoTensor`.

**Status:** pre-alpha (`0.0.0`). API is in flux; no operators have been implemented yet.

## Installation

Not yet on PyPI. Install from source:

```bash
git clone https://github.com/jejjohnson/geotoolz.git
cd geotoolz
make install
```

## Quickstart

```python
import geotoolz
```
